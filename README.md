# PyTorch Learning Sandbox

A personal learning journey through machine learning and deep learning concepts using PyTorch and PyTorch Lightning.

## Overview

This repository documents my hands-on exploration of ML/DL fundamentals, from basic tensor operations to complete deep learning models. It serves as a practical showcase of my learning process and experience with modern PyTorch tools.

## Learning Path

### 1. **Basics**
- Tensor fundamentals and operations
- Computation graphs and autograd

### 2. **Neural Network Layers**
- Linear layers and activation functions
- Normalization, dropout, and embedding layers

### 3. **Machine Learning Models**
- Linear regression implementations
- Logistic regression for classification

### 4. **Deep Learning**
- Custom deep learning models
- PyTorch Lightning integration for production-ready training

## Purpose

This is primarily a learning showcase rather than a production library. Each implementation focuses on understanding core concepts rather than optimization or deployment.

## Tools & Technologies
- **PyTorch** - Core deep learning framework
- **PyTorch Lightning** - Professional training workflows
- **Matplotlib** - Visualization
- **scikit-learn** - Data utilities
